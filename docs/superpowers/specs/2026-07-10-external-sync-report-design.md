# 外发同步版双周报设计

## 目标

把教育行业双周报从“投资部研究任务展示”调整为“面向领导和同事同步本期行业进展”的外发材料，同时保留投资部内部研究所需的数据字段。

## 页面口径

1. 第二页由“本期核心观点”改为“本期主要动态”。
2. 主要动态按现有栏目分组，自动摘取本期正式事件标题，采用截图中的“栏目标题 + 事项列表”结构，不展示证据编号或投资判断式结论。
3. 融资与交易有合格事项时仍排在最前；空栏目不展示。
4. 事件页保留事实、主体背景、行业判断、受益方和风险，不展示“后续跟踪”。
5. 末页改为“本期行业小结”，只保留对本期变化的概括，不展示“下一期验证”。

## 内外分离

`tracking` 继续作为内部事件研究卡字段，由校验器检查完整性，但 PowerPoint、逐页稿和对外质量表述不得渲染该字段。`core_insights` 继续用于内部分析一致性和行业小结生成，但不直接形成对外页面。

## 生成逻辑

生成器从 `sections` 中读取非空栏目，去掉“行业速览-”前缀后形成栏目标题；每个栏目按原顺序展示事件 `headline`。第二页最多展示 8 个正式事项，超出时按五维总分和原顺序选取，不得用检索结论或跨期事项补位。

事件页删除跟踪框后扩大行业判断区域，避免形成空白断层。结尾页扩大行业小结区域，并显示“本期共纳入 N 项经核验行业动态”的客观说明。

## 验收标准

- 第二页标题为“本期主要动态”，含融资与交易及其他非空栏目。
- 完整 PPT 不含“本期核心观点”“证据事项”“后续跟踪”“下一期验证”。
- `tracking` 仍存在于输入契约和内部校验中。
- 融资页仍位于政策页之前。
- 教育与职教两个生成器行为一致。
- 当前双周报重新生成后通过完整测试、10页渲染和溢出检查。
