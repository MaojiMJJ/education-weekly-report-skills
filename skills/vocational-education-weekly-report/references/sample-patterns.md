# 样例页式规律

样例来源：本项目已下载 PDF：

- `职教行业周报2026.05.11-2026.05.31.pdf`
- `职教行业周报2026.06.15-2026.06.28.pdf`

## 观察到的结构

- 封面页：标题 `职教行业周报` + 日期区间。
- 内容页：每页一个事项。
- 顶部或左上角显示序号和栏目，例如 `1 职业教育-政策`。
- 左侧或标题区使用短标题，正文为项目符号。
- 典型页数：封面 + 5-7 个内容页。
- 常见栏目顺序：政策优先，其次职业学院，再其他公司 / 产教融合 / 融资事项。

## 样例事项类型

- 政策：国务院、教育部、省级条例、地方产教融合法规。
- 职业学院：职业技术大学设置、公办本科层次职业学校、首批职业本科专业。
- 其他：行业产教融合共同体成立、职教共同体联盟、非上市企业融资。

## 写作尺度

- 每个事项通常 2-5 个要点。
- 第一条说明新闻事实。
- 后续条说明文件内容、业务内容、参与方、规模或影响。
- 不写长篇背景，除非该背景解释事项重要性。
