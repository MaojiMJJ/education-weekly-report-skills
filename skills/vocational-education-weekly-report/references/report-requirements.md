# 职教行业周报要求

## 交付形态

- 主要形式：PPT。
- 频率：每两周一次。
- 页面规则：每页一个事项，每页包含标题和正文内容。
- 常用标题：`职教行业周报`。

## 核心关注事项

1. 新设校企合作产业学院：必须说明合作校企双方分别是谁、合作内容是什么。
2. 职业学院、本科院校最新资讯：包括院校新成立、合并、停止办学，新设专业或二级学院，独立学院转设，职业学院升本。
3. 职教政策：发布机关包括教育部及各省市教育厅、教育局。
4. 职教赛道上市公司：新业务、新合作、新产品、并购事件、投资事件。
5. 职教赛道非上市企业：新业务、新合作、新产品、投融资事件。

## 推荐栏目

- `职业教育-政策`
- `职业教育-职业学院`
- `职业教育-产业学院`
- `职业教育-上市公司`
- `职业教育-融资与交易`
- `职业教育-其他`

## 字段要求

### 政策

- 发布机关。
- 文件或会议名称。
- 发布时间和实施时间。
- 与职业教育直接相关的条款。
- 对学校、企业、产教融合、技能培训、专业设置或职业本科的影响。

### 职业学院 / 本科院校

- 学校名称和所在地。
- 事件类型：新设、合并、停止办学、升本、专业设置、二级学院、转设。
- 批准或发布机关。
- 规模信息：专业、学生规模、首批专业、合作方等。
- 对区域职业教育或产业人才培养的意义。

### 产业学院 / 校企合作

- 学校方。
- 企业方。
- 合作内容：专业共建、课程、实训基地、订单班、现代学徒制、产业学院、共同体等。
- 面向产业或岗位。
- 是否有资金、基地、招生或培养规模披露。

### 上市公司

- 股票代码、公司简称。
- 事件类型：新业务、新合作、新产品、并购、投资。
- 交易或合作对象。
- 交易金额、股比、估值、支付方式等已披露信息。
- 公司主营业务和最近业绩简况。
- 对职业教育业务布局的意义。

### 非上市企业 / 融资

- 公司名称、成立时间、所在地。
- 细分赛道和客户类型。
- 产品或服务形态。
- 融资轮次、金额、币种、投资方。
- 业务进展：用户数、学校数、门店数、收入、合作案例等已披露信息。

## 排除边界

- K12、国际教育、素质教育、高等教育、教育信息化等非职教事项，除非事件直接指向职业教育、职业院校、技能培训或产教融合。
- 泛教育上市公司普通资本动作，如果与职业教育无关，不纳入本周报。
