# 事件研究卡模板

## 事实门槛

评分前先确认：日期、主体、动作、来源均明确；重大事项有一手来源；关键事实可由来源逐条核验。

## 五维评分

每项 1-5 分，并写事实理由：

| 维度 | 1 分 | 3 分 | 5 分 |
| --- | --- | --- | --- |
| 行业影响 | 单一活动或日常信息 | 影响一所学校或一个项目 | 改变职教规则、办学供给或企业需求 |
| 政策重要性 | 无政策关系 | 地方倡议或一般试点 | 国家或省级资金、名单、验收和强制要求 |
| 商业价值 | 只有挂牌或签约 | 有投入、招生或客户 | 收入、订单、培训人数或规模化路径明确 |
| 技术相关性 | 无专业或技术内容 | 对接一个产业或专业 | 改变实训、教学、评价或岗位技能标准 |
| 投资相关性 | 无法量化 | 可形成跟踪假设 | 交易、合同、业绩、投入或市场空间可验证 |

入选条件：总分不低于 16；或行业影响不低于 4，且其余维度至少一项不低于 4。

```text

- 学校、专业、企业、产业、客户、地区、历史合作或最新业绩。

行业判断：
- 改变了什么变量？
- 代表什么趋势？
- 为什么在当前报告期重要？

```

## 判断约束

- 行业判断必须包含因果链，不能只是把事实换一种说法。
- “受益”要指出受益机制，“风险”要指出失败条件。
- 跟踪问题必须可验证，例如招生、专业、设备、企业投入、岗位、就业、合同、回款或验收时间。
- 未披露字段写入底稿，不在正文补造。
