# 信息源和检索流程

## 主要信息源

- 多知网：独立商业视角，新锐教育观察。
- 芥末堆：教育产业资讯。
- 蓝鲸财经教育频道。
- 多鲸资本。
- 黑板洞察：教育产业数据研究。
- 教育部官网：中华人民共和国教育部政府门户网站。
- 海淀教委官方公众号：海淀教育。
- 上市公司公告：巨潮资讯网。

## 优先级

1. 官方文件、教育部、省市教育厅、教育局、学校官网、上市公司公告、交易所公告、巨潮资讯。
2. 公司官网、官方公众号、投资方公告。
3. 教育产业媒体和财经媒体。
4. 二次转载、论坛、未经确认的自媒体。

当媒体报道与官方来源冲突时，以官方来源为准。没有官方确认的交易金额、合作内容或政策解读不得写成确定事实。

## 建议检索词

- `职业教育 政策 教育部`
- `职业教育 条例 省 教育厅`
- `职业本科 设置 公示`
- `职业技术大学 设置`
- `产教融合 共同体 成立`
- `产业学院 职业院校 企业 合作`
- `校企合作 产业学院 职业教育`
- `职业教育 融资`
- `职业培训 融资`
- `技能培训 投资`
- `传智教育 职业教育 合作`
- `中公教育 职业教育 公告`
- `中国东方教育 投资 合作`

## 检索步骤

1. 先用时间窗检索：报告期起止日期加上 1-3 天缓冲。
2. 分栏目检索：政策、职业学院 / 职业本科、产业学院、上市公司、非上市公司融资。
3. 用上市公司清单逐一检索重大公告和业务合作。
4. 对入选候选项补官方来源；如果只有媒体报道，至少找到第二个独立来源或说明信息披露有限。
5. 记录 source log：标题、URL、发布方、发布日期、访问日期、对应 slide。

## 入选标准

- 与职业教育直接相关。
- 发生在报告期内，或在报告期内披露且有新增事实。
- 能回答“发生了什么、涉及谁、为什么重要”。
- 对职业教育政策、学校设置、产教融合、上市公司布局或企业融资有行业观察价值。

## 剔除标准

- 只有活动宣传、会议合影，没有实质合作内容。
- 没有金额、主体、产品、政策条款等关键事实的融资或合作传闻。
- 与职业教育无关的泛教育新闻。
- 同一事项重复报道，只保留事实最完整的一条。
