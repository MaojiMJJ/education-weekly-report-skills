# 信息源和检索流程

## 资料边界

先确认用户允许使用哪些资料：

- 公开来源：政府网站、学校官网、企业官网、交易所公告、行业媒体公开网页。
- 用户文件：用户明确提供的 Word、PDF、Excel 或其他材料。
- 邮箱材料：只有用户明确允许时才能使用。用户禁止邮箱时，不读取、不摘要、不引用。

## 主要信息源

优先顺序如下：

1. 官方文件和公告：教育部、国务院、省市教育厅、省市教育局、学校官网、上市公司公告、交易所公告、巨潮资讯。
2. 企业和投资方官方来源：公司官网、官方公众号、投资方公告。
3. 教育行业媒体：多知网、芥末堆、蓝鲸财经教育频道、多鲸资本、黑板洞察。
4. 财经媒体和地方媒体。
5. 二次转载、自媒体、论坛内容只能作为线索，不能作为最终事实来源。

当媒体报道与官方来源冲突时，以官方来源为准。

## 必跑检索模块

每期至少执行以下模块，并记录结果：

### 政策

检索词：

- `职业教育 政策 教育部`
- `职业教育 条例 省 教育厅`
- `产教融合 职业教育 政策`
- `职业本科 教育厅`
- `技能培训 就业 职业教育`

### 职业学院和职业本科

检索词：

- `职业技术大学 设置`
- `职业本科 批准`
- `职业学院 升本`
- `职业教育 专业目录`
- `职业院校 新设专业`

### 产业学院和校企合作

检索词：

- `产业学院 职业院校 企业 合作`
- `校企合作 产业学院 职业教育`
- `产教融合共同体 成立`
- `行业产教融合共同体 职业院校`

### 上市公司

先读取 `listed-companies.md`，再逐家公司检索：

- 公司简称 + `公告`
- 公司简称 + `职业教育 合作`
- 公司简称 + `投资`
- 公司简称 + `并购`
- 公司简称 + `新业务`

### 非上市公司和融资

检索词：

- `职业教育 融资`
- `职业培训 融资`
- `技能培训 投资`
- `产教融合 融资`
- `职业教育 新产品`

## 候选池字段

每条候选事项必须记录：

- 标题。
- 分类。
- 发布日期。
- 来源名称。
- URL。
- 事项主体。
- 关键事实。
- 已披露字段。
- 缺失字段。
- 入选或剔除理由。

## 入选标准

同时满足：

- 与职业教育直接相关。
- 发生在报告期内，或报告期内有关键新增披露。
- 能回答“发生了什么、涉及谁、为什么重要”。
- 有公开可访问来源。

## 生成前检查

- 事项数不足 5 个时，先输出检索底稿和缺口说明。
- 上市公司事项必须有公告或公司官方来源。
- 院校设置、升本、专业设置必须有教育主管部门或学校官网来源。
- 融资事项至少要有公司、轮次、金额或投资方中的两个核心字段。
